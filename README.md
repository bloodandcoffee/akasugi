# 赤杉 akasugi

**akasugi** is a simple Japanese input method for Wayland compositors

[![GPLv3](https://img.shields.io/badge/license-GPLv3-green)](#)

</div>


### USAGE

    sudo akasugi

### DEPENDENCIES

- libevdev

### INSTALLING

Compile

    make

Install

    make install

Uninstall

    make remove
